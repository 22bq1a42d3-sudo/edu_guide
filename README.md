🎓 Edu Guide

📌 Lack of Career Guidance for School Students

Edu Guide is an AI-based application that helps school students choose the right career path by providing personalized guidance and recommendations.

---

🚀 Features

- Career suggestions based on interests
- AI-powered guidance
- Simple and user-friendly interface

---

🛠 Tech Stack

- Flutter (Frontend)
- Node.js / Python (Backend)
- OpenAI API

---

▶️ How to Run

1. Clone the repo
2. Install dependencies
3. Add ".env" file
4. Run the project
